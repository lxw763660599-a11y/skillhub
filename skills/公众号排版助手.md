# 公众号排版助手

一键将 Markdown 内容转换为符合微信公众号阅读习惯的精美排版。

## 功能

- **多主题切换**：科技风、文艺风、商务风、极简风
- **自动格式优化**：段落间距、字体颜色、引用样式、分割线
- **代码高亮**：支持 20+ 编程语言语法高亮
- **图片自动适配**：自动调整为微信最优宽度

## 使用方法

```
用公众号排版助手，帮我把下面的内容排版成科技风：

[粘贴你的 Markdown 内容]
```

## 示例

输入：
```markdown
# 2026年AI行业5大趋势

## 1. Agent 全面落地
企业级AI Agent将从实验阶段进入大规模部署...
```

输出：自动生成带分割线、引用框、emoji装饰、适当留白的微信图文。

## 适用场景

- 技术博客转公众号文章
- 产品发布文案
- 教程和指南
- 行业分析报告

## 技巧

- 在 prompt 里指定风格关键词：温暖、专业、活泼、极简
- 需要配图时，告诉它"为每个小节配一张示意插图"
- 超过3000字的内容建议分段处理
