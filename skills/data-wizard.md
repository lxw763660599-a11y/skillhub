# 数据分析引擎

上传原始数据，自动完成从数据清洗到分析报告的全流程。

## 功能

- **数据清洗**：缺失值处理、异常值检测、格式标准化、去重
- **探索性分析**：分布可视化、相关性矩阵、特征重要性
- **统计建模**：回归分析、聚类、时间序列预测
- **AB测试分析**：显著性检验、效应量计算、置信区间
- **报告生成**：自动撰写分析报告，含图表和结论

## 使用方法

```
分析这份用户行为数据：[上传 user_behavior.csv]

我想要：
1. 用户留存曲线
2. 付费转化漏斗分析
3. 高价值用户特征画像
4. 流失预警模型
```

## 输出示例

```
## 核心发现
- 次日留存率 45%，7日留存 28%，低于行业平均
- 付费转化率仅 3.2%，漏斗最大流失在"添加购物车→支付"
- 高价值用户特征：iOS用户、使用超过7天、完成3次以上搜索

## 建议
1. 优化支付流程，预计可提升转化至 5%+
2. 引导新用户完成首次搜索，7日留存可提升至 35%
```

## 适用场景

- 用户运营分析
- 财务数据建模
- 市场调研数据处理
- 学术研究数据分析

## 技巧

- 给数据前先描述每列的含义
- 不清楚分析方法时让它推荐
- 要求"给出业务建议，不只是数据描述"
