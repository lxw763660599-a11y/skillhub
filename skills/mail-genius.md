# AI 邮件助手

写邮件、回邮件、总结邮件线程，各种商务邮件场景全覆盖。

## 功能

- **智能撰写**：给要点自动扩充为正式邮件
- **邮件回复**：根据上下文自动生成得体回复
- **线程总结**：20封邮件线程，3句话总结核心
- **语气调节**：正式/商务/亲切/催促/道歉
- **多语言**：中英日韩商务邮件无缝切换
- **附件提取**：自动提取并整理附件信息

## 使用方法

```
帮我在这个邮件线程上回复：[粘贴邮件内容]

我的态度：需要更多时间评估，但方向认可
语气：商务礼貌
```

```
帮我写一封邮件给客户：
- 通知产品上线延期1周
- 解释原因（需等待安全审查）
- 表达歉意并承诺质量
- 提供备选时间线
语气：诚恳但专业
```

## 邮件模板库

| 场景 | 关键要素 |
|------|----------|
| 商务合作 | 自我介绍 + 合作价值 + 下一步 |
| 项目延期 | 坦诚原因 + 新时间线 + 补救方案 |
| 催款/催进度 | 事实陈述 + 明确期限 + 柔和语气 |
| 拒绝请求 | 感谢 + 拒绝理由 + 替代方案 |
| 会议邀约 | 目的 + 时间段选项 + 议程 |

## 技巧

- 给它你之前的邮件，它会学习你的写作风格
- 长邮件线程用"3句话总结"功能快速跟上
- 英文商务邮件加一句"地道商务英语，不要太正式"
