# PPT 一键生成

输入内容大纲或文档，自动生成结构清晰的 PPT 文件。

## 功能

- **大纲转 PPT**：3 级大纲自动生成页面结构
- **文档转 PPT**：Word/Markdown 文档直接转化
- **多风格模板**：商务蓝、科技黑、清新绿、创意彩
- **自动配图**：每页智能匹配示意插图
- **数据图表**：自动将数据转为柱状图/饼图/折线图
- **演讲备注**：自动生成每页演讲要点

## 使用方法

```
帮我做一个工作汇报PPT，内容如下：
- 上季度业绩回顾
- 重点项目进展
- 存在问题与改进
- 下季度计划

风格选商务蓝，10页左右
```

## 最佳实践

**好的输入 → 好的输出：**
```
主题：2026年Q2产品发布会
受众：投资人
风格：科技感深色
页数：15页
重点：3个新功能demo、市场数据、竞争分析、融资计划
```

**烂输入 → 烂输出：**
```
"做个PPT"
```

## 适用场景

- 工作周报/月报
- 产品发布会
- 项目立项/结项汇报
- 培训课件
- 商业计划书

## 技巧

- 提供越详细的大纲，生成质量越高
- 指定"每页不超过5个要点"
- 数据部分直接给数字，会自动生成图表
