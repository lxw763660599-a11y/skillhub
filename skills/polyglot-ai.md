# 多语言翻译引擎

高质量多语言翻译，保留原文语气、格式和专业术语，支持整篇文档翻译。

## 功能

- **50+语言**：中、英、日、韩、法、德、西、阿、俄等
- **格式保留**：Markdown/HTML/JSON 格式无损
- **术语一致性**：上传术语表，确保专有名词翻译一致
- **语气适配**：正式/商务/口语/营销文案
- **代码安全**：自动跳过代码块不翻译

## 使用方法

```
把这篇技术文档翻译成英文：[粘贴内容]

要求：
- 保留所有 Markdown 格式
- 专业术语参考附件的术语表
- 代码块不要翻译
```

## 输出对比

**机翻味翻译：**
> "在我们的产品中，我们实现了最好的用户体验"

**本引擎翻译：**
> "Our product delivers a best-in-class user experience through..."

## 适用场景

- 技术文档国际化
- 产品文案本地化
- 商务邮件翻译
- 论文摘要翻译
- App/网站多语言包

## 技巧

- 术语表格式：`中文 → English → 备注`
- 指定目标读者背景（技术/商务/普通用户）
- 让它"翻译完再回译检查一遍"
