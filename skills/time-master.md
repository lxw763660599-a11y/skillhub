# 智能日程管理

输入你的待办事项和时间约束，自动排出最优日程表，并智能处理冲突。

## 功能

- **自动排程**：根据优先级、预估时长、截止日期自动排列
- **冲突检测**：时间冲突自动提醒并提供替代方案
- **番茄钟集成**：大任务自动拆分为番茄钟时段
- **周报生成**：自动汇总本周完成情况
- **习惯养成**：周期性任务自动提醒

## 使用方法

```
帮我安排明天的工作：

待办：
1. 完成Q2报告（3小时，下午5点前提交）🔴
2. 面试候选人（1小时，10:00-11:00已定）🔴
3. 回复客户邮件（30分钟）🟡
4. 整理报销单据（20分钟）🟢
5. 健身（1小时）🟢

明天9点上班，6点下班，中午12-13点午休
```

## 输出示例

```
📅 2026年6月2日 日程安排

09:30-10:00  整理报销单据 🟢
10:00-11:00  面试候选人 🔴 [已定]
11:00-12:00  Q2报告 Part1 🔴
12:00-13:00  午休 🍜
13:00-15:00  Q2报告 Part2 🔴
15:00-15:30  缓冲时间
15:30-16:00  Q2报告复查提交 🔴
17:00-18:00  健身 🟢
```

## 适用场景

- 日常工作安排
- 项目冲刺阶段规划
- 多人会议协调
- 考试复习计划

## 技巧

- 用🔴🟡🟢标记优先级更直观
- 多人日程让它"找出3人都有空的时间段"
- 告诉它你的高效时段（比如上午效率高放重要任务）
