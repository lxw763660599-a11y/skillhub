# Excel 智能分析

上传一个 Excel 文件，自动完成数据清洗、统计分析、可视化图表和洞察报告。

## 功能

- **自动数据概览**：行数、列数、数据类型、缺失值检测
- **统计分析**：均值、中位数、分布、相关性矩阵
- **智能图表**：自动选择最佳图表类型（柱状图、折线图、散点图、热力图）
- **异常检测**：标记离群值和异常数据点
- **洞察报告**：用自然语言总结关键发现

## 使用方法

```
帮我分析这份销售数据 [上传 sales.xlsx]

我想知道：
1. 各地区的销售额排名
2. Q2同比变化趋势
3. 是否存在异常订单
```

## 示例输出

- 各地区销售额柱状图
- Q2月度趋势折线图
- 异常订单列表及原因分析
- 一句话总结：华东区销售额同比增长23%，但华南区7月出现异常下降

## 适用场景

- 销售数据分析
- 财务报表解读
- 运营数据跟踪
- 用户行为分析
- 库存管理
