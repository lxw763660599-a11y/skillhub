# 会议纪要生成

上传会议录音或记录文本，自动生成清晰的结构化会议纪要。

## 功能

- **语音转文字**：支持中文/英文/中英混合录音
- **智能摘要**：提取关键讨论点和决策结论
- **待办提取**：自动识别并列出每个人的 Action Items
- **时间线**：按时间顺序整理讨论脉络
- **多格式导出**：Markdown / 飞书文档 / 企业微信

## 使用方法

```
帮我整理这次会议纪要：[上传 meeting.mp3]

格式要求：
- 参会人
- 讨论议题及结论
- 待办事项（标注负责人和截止日期）
- 下次会议时间
```

## 输出示例

```markdown
# 产品评审会纪要
**时间**：2026-06-01 14:00-15:30
**参会人**：张三、李四、王五

## 讨论结论
1. Q3版本增加离线模式 → 全票通过
2. UI改版延至Q4 → 资源不足

## 待办事项
- [ ] 张三：离线模式技术方案 → 6月5日前
- [ ] 李四：UI改版排期评估 → 6月8日前
- [ ] 王五：用户调研问卷发布 → 6月3日前
```

## 适用场景

- 产品评审会
- 周会/站会
- 客户访谈
- 项目复盘会
- 面试记录

## 技巧

- 多人录音告诉它发言人身份，识别人名更准
- 长会议（>1小时）建议"只提取结论和待办"
- 中英混用会议加一句"会议语言为中英混合"
