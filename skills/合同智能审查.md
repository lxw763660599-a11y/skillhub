# 合同智能审查

上传合同 PDF 或 Word 文件，自动审查条款风险，生成修改建议。

## 功能

- **条款完整性检查**：识别缺失的关键条款（违约责任、争议解决、保密条款等）
- **风险标记**：自动标红不平等条款、模糊表述、金额歧义
- **行业对比**：对照行业标准合同，发现异常条款
- **修改建议**：逐条给出修改方案和替代措辞
- **摘要报告**：3分钟速读版合同摘要

## 使用方法

```
帮我审查这份服务合同：[上传 contract.pdf]

重点看：
1. 违约责任是否对等
2. 知识产权归属
3. 付款条件是否合理
```

## 审查维度

| 维度 | 检查项 |
|------|--------|
| 主体信息 | 名称、地址、统一信用代码 |
| 价款条款 | 金额、税率、支付节点 |
| 交付标准 | 验收条件、期限、延期罚则 |
| 违约责任 | 是否对等、上限是否合理 |
| 知识产权 | 归属、许可范围、侵权责任 |
| 保密条款 | 范围、期限、例外 |
| 争议解决 | 管辖法院、仲裁条款 |

## 适用范围

- 采购合同
- 技术服务合同
- 劳动合同
- 租赁合同
- 保密协议 (NDA)

## 重要提示

本工具提供参考建议，不构成法律意见。重大合同请咨询专业律师。
