# 社交媒体运营

输入一个话题或产品，生成适合不同平台的社交媒体内容。

## 功能

- **多平台适配**：小红书/抖音/微博/知乎/公众号/推特/LinkedIn
- **风格切换**：种草/干货/故事/热点/教程
- **话题标签**：自动推荐热门标签和关键词
- **发布时间建议**：根据不同平台用户活跃时段推荐发布时间
- **A/B 测试**：同一主题生成多个版本对比

## 使用方法

```
帮我为这款时间管理App写社交媒体内容：

产品：FocusTime，番茄钟+任务管理
本周话题：远程办公效率提升

请生成：
1. 小红书种草文案（带emoji和标签）
2. 知乎干货回答框架
3. 微博短文案（含话题标签）
```

## 平台格式

| 平台 | 字数 | 关键元素 |
|------|------|----------|
| 小红书 | 300-500字 | 封面标题 + emoji + #标签# + 品牌名称 @ |
| 知乎 | 800-1500字 | 开头金句 + 分段干货 + 结尾互动 |
| 微博 | 140字内 | #话题# + @相关账号 + 配图 |
| 公众号 | 1000-3000字 | H2分级 + 引用 + 分割线 + 引导关注 |

## 技巧

- 给竞品爆款链接，让它分析并模仿风格
- 指定"怼人""幽默""温暖"等调性
- 批量生成一周的内容日历
