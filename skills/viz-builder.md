# 数据可视化仪表盘

上传数据文件，自动生成交互式可视化仪表盘，支持实时筛选和下钻。

## 功能

- **自动图表选择**：根据数据类型智能选择柱状图、折线图、饼图、散点图、热力图
- **多图表联动**：点击一个图表的数据，其他图表自动联动筛选
- **下钻分析**：从年度→季度→月度逐层深入
- **多种导出**：PNG / SVG / PDF / HTML
- **暗色模式**：大屏展示优化

## 使用方法

```
用这份销售数据帮我做一个仪表盘：[上传 sales.csv]

包含：
- 各区域销售额地图
- 产品类别占比饼图
- 月度趋势折线图
- TOP 10 客户柱状图
- 关键指标卡片（总销售额、同比增长率、客单价）
```

## 适用场景

- 管理驾驶舱
- 融资 pitch deck 数据页
- 运营日报/周报
- 数据大屏
- 分析报告插图

## 技巧

- 数据第一行必须是列名
- 告诉它"你想从数据中看出什么"，它会选对图表
- 做融资用的图表指定"加粗关键数字，用品牌色"
