# 沉浸式写作助手

极简写作体验 + AI 智能辅助，专注内容创作本身。

## 核心功能

- **无干扰模式** — 隐藏 UI 元素，沉浸式码字
- **实时语法纠错** — 错别字、语病即时标注
- **AI 续写** — 卡壳时帮你接下去
- **风格优化** — 调整语气（正式/口语/幽默）
- **多格式导出** — Word / PDF / HTML / 公众号
- **番茄钟** — 内置专注计时

## 使用方式

```bash
# 启动写作环境
write-flow start my-article.md

# AI 续写
write-flow continue "接下来讨论..."

# 调整风格
write-flow polish --style formal document.md

# 导出
write-flow export article.md --format pdf
```

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+J | AI 续写 |
| Ctrl+K | 风格优化 |
| Ctrl+P | 导出 |
| F11 | 全屏无干扰 |
