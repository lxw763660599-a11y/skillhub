# SEO 文章写作

给一个核心关键词，自动生成 SEO 友好的长文章，包含标题优化、内链、关键词布局。

## 功能

- **关键词研究**：根据主关键词扩展长尾词和 LSI 关键词
- **标题优化**：生成 5 个高点击率标题备选
- **结构化写作**：H1→H2→H3 层级清晰，利于搜索引擎抓取
- **关键词布局**：关键词自然分布在首段、小标题、尾段
- **元描述**：自动生成 150 字以内的 meta description
- **内链建议**：推荐相关文章作为内链

## 使用方法

```
帮我写一篇SEO文章，核心关键词：远程办公工具推荐

要求：
- 字数：2000+
- 目标受众：中小企业和自由职业者
- 包含3-5款工具的对比表格
- 自然地融入关键词"远程办公""效率工具""团队协作"
```

## 输出结构

```
# [原标题]
[引言 - 200字，含核心关键词]

## [第一个H2小标题]
[正文 - 300字]

## [第二个H2小标题]
[正文 - 300字]

...

# [总结]
[结尾 - 含CTA]
```

## 适用场景

- 企业博客内容营销
- 独立站 SEO 优化
- 知乎/小红书长文
- 产品使用教程

## 技巧

- 提供竞品文章链接，让它分析并写出更好的
- 指令中加入"像真人写作，避免AI感"
- 指定文章调性：专业/轻松/实操/深度
