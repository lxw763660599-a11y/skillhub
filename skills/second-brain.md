# 个人知识库

自动整理你的笔记、收藏夹、文档和聊天记录，构建可搜索的个人知识系统。

## 功能

- **自动分类**：AI 自动识别内容主题并归类
- **智能关联**：自动发现相关内容之间的关联
- **全文搜索**：支持中文语义搜索，模糊匹配
- **知识图谱**：可视化展示知识之间的关联网络
- **定期回顾**：基于遗忘曲线的智能复习提醒
- **多源接入**：笔记App/浏览器书签/聊天记录/PDF/网页

## 使用方法

```
帮我把最近收集的关于"用户增长"的资料整理成知识库：

[粘贴网页链接、笔记、PDF内容]

按以下结构组织：
1. 核心概念
2. 方法论
3. 工具清单
4. 案例库
5. 待实践清单
```

## 知识图谱示例

```
用户增长
├── 获客渠道
│   ├── SEO → [相关笔记3篇]
│   ├── SEM → [案例2个]
│   └── 内容营销 → [文章5篇]
├── 激活转化
│   └── A/B测试 → [工具对比表]
└── 留存
    ├── 推送策略 → [最佳实践]
    └── 用户分层 → [分析模板]
```

## 适用场景

- 个人学习笔记管理
- 团队知识沉淀
- 项目文档归档
- 读书笔记整理

## 技巧

- 先给它你的知识分类体系，之后自动归类
- 每条笔记加标签，关联检索更准
- 定期让它"发现知识盲区"
