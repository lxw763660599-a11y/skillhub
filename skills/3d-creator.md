# 3D 模型生成

文字描述或上传参考图，自动生成 3D 模型，支持多种格式导出。

## 功能

- **文生3D**：输入描述文字直接生成3D模型
- **图生3D**：上传图片生成对应的3D模型
- **多格式导出**：GLB/OBJ/FBX/STL/USDZ
- **PBR材质**：自动生成逼真的材质和纹理
- **模型优化**：减面、展UV、LOD自动生成

## 使用方法

```
生成一个北欧极简风格的沙发3D模型：
- 浅灰色布艺
- 木质细腿
- 三人座
- 导出为GLB格式
```

```
把这张产品照片转成3D模型：[上传 product.jpg]
用于AR展示，导出USDZ格式
```

## 适用场景

- 电商AR展示
- 游戏资产快速制作
- 室内设计预览
- 3D打印模型
- 产品原型展示

## 技巧

- 描述越具体越好：尺寸、材质、颜色、风格
- 图生3D用正视角清晰照片效果最好
- 3D打印用指定"封闭网格，无破面"
