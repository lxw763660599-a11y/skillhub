# 网页内容提取器

输入 URL，自动提取网页正文、关键数据和结构化信息。

## 功能

- **正文提取**：去除广告、导航栏、侧边栏，只保留正文
- **表格提取**：自动识别并转为 CSV/Markdown 表格
- **列表提取**：提取产品列表、文章列表等
- **元数据提取**：标题、发布时间、作者、关键词
- **多页面批量**：支持列表页 → 详情页的批量抓取

## 使用方法

```
帮我把这篇文章的内容提取出来：https://example.com/article

提取成 Markdown 格式，保留原标题和发布时间
```

批量使用：
```
这个列表页有20篇文章：https://example.com/list
帮我提取每篇的标题、摘要、链接，输出成表格
```

## 适用场景

- 竞品分析（批量提取产品信息）
- 内容采集（文章、新闻）
- 数据迁移（网页表格 → Excel）
- 舆情监控（评论、评价提取）

## 技巧

- 需要登录的页面让它用浏览器自动化打开
- 动态加载的页面用"等待5秒再提取"
- 提取评论时指定"按评分从高到低排序"
