# 短视频脚本生成

输入主题和产品信息，自动生成短视频脚本，包含分镜、口播文案和拍摄建议。

## 功能

- **脚本结构**：黄金3秒开头 + 痛点/爽点 + 产品展示 + CTA结尾
- **分镜描述**：每个镜头画面、时长、台词、字幕
- **风格适配**：口播、vlog、教学、剧情、产品测评
- **平台优化**：抖音/小红书/视频号/B站定制
- **文案提取**：自动提取金句作为视频标题和描述

## 使用方法

```
帮我写一个抖音短视频脚本，推广这款AI笔记工具：

产品卖点：
- 语音转文字，准确率98%
- 自动生成思维导图
- 一键分享给团队

目标用户：职场白领
时长：60秒
风格：轻松幽默
```

## 输出示例

```
【分镜1】3秒
画面：凌晨1点，办公室里只有一个人在打字
口播："凌晨1点还在整理会议纪要？你缺的不是咖啡，是这个"
字幕：凌晨1点，你还在...

【分镜2】5秒
画面：手机靠近说话人，文字实时弹出
口播："开会时打开它，老板说的话自动变成文字和思维导图"
...
```

## 适用平台

- 抖音（竖屏9:16）
- 小红书（种草风格）
- 视频号（偏专业）
- B站（中长视频）

## 技巧

- 给竞品视频链接，让它模仿风格
- 指定BGM风格（激昂/轻松/悬疑）
- 多人出镜时描述每个人的角色
